# Web Development Playground

Repositório dedicado à prática e consolidação de fundamentos em HTML, CSS e JavaScript.

---

## Sobre

Este repositório reúne exercícios e implementações desenvolvidas ao longo dos estudos, com foco na construção de uma base sólida em desenvolvimento web.

O conteúdo prioriza organização, clareza de código e aplicação consistente de conceitos fundamentais.

---

## Tecnologias

* HTML5
* CSS3
* JavaScript (ES6+)

---

## Estrutura

A organização do repositório segue uma separação simples por responsabilidade:

* `/html` — estrutura e semântica
* `/css` — estilização e layouts
* `/js` — lógica e manipulação de DOM

---

## Objetivo

* Consolidar fundamentos de desenvolvimento web
* Aprimorar raciocínio lógico
* Manter um padrão de código limpo e organizado

---

## Observações

Este repositório é continuamente atualizado conforme a evolução dos estudos.
